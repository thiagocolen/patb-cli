Welcome to the official documentation site for **`patb-cli`** — the interactive CLI and
Zed ACP bridge for the remote "Pinky and the Brain" agent service.

<!-- truncate -->

This site covers everything you need to get started and to dig into how the CLI works
under the hood:

- **End-User Documentation** — installation, configuration, day-to-day usage of the
  interactive REPL and the Zed bridge, a command reference, and an FAQ /
  troubleshooting knowledge base.
- **Developer & Technical Documentation** — project structure, architecture, a
  module-by-module source code reference, and the ACP protocol implemented by the
  bridge.

Head over to the [Introduction](/docs/intro) to get started, or jump straight to
[Installation](/docs/end-user/installation).
